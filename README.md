# yapm
## yet another package manager

YAPM is a package manager (duh) aimed to be an all-in-one tool for package management across all platforms (by parseing their seperate package managers).

I've made it open source. I will accept and HAPPILY take any contributions. You can also use this repo to host your own packages by submitting a pull request with your package by following the instructions in /yapm-package-format.
